sysSetEnvironment¶

function (sysSetEnvironment unid shape options) → True/Nil¶
Parameters:

shape

'arc

Creates an arc along an orbit. Options must include the following fields:

'orbit: This is the orbit to use. The arc will be

centered on the current orbit position (e.g., the planet location).

'length: The length of the arc (in degrees). 'width: The width at the center of the arc (in light-

seconds).

'orbital

Creates a random environment along the orbit. Options must include the following fields:

'orbit: This is the orbit to use. 'width: The average width of the ring, in light-

seconds.

'square
Creates a square patch. Options must include the

following fields:

'center: The center position of the patch. 'height: The height of the patch (in light-seconds). 'width: The width of the patch (in light-seconds).
